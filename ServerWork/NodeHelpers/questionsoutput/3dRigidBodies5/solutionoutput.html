    Now that we identified the reaction forces we can now start to solve the problem.

    First, we want to represent the tension in the beam as a vector.

    We can do this by simply multiplying the tension force by the unit vector that follows the same path as the beam.

    So, let the unit vector goes from point D to C be \( \vec{U} = \frac{L_3\hat{i}+L_2\hat{j}+L_1\hat{k}}{\sqrt{(L_3)^2+(L_2)^2+(L_1)^2}} \)

    Now that we have the unit vector we can say the tension vector is:

  • \( \vec{T} = (T)\vec{U} = (T)\frac{L_3\hat{i}+L_2\hat{j}+L_1\hat{k}}{\sqrt{L_3^2+L_2^2+L_1^2}} \).
