Level 0

The first step is to draw a free body diagram. And write ou the impulese moment equation.

Level 1

Using the impulse momentum equation in the horizontal direction:

  • \( mv_i + \Sigma \int F_x dt = m v_f \)
  • \( 0 + (Fcos(\theta) t - \mu_k N t ) = m v_f \)

    • The block starts at rest so \( v_i \) is 0. We can solve for F.

    Level 2

      Solving for F:

    • \( (Fcos(\theta) t - \mu_k N t ) = m v_f ~\implies~ F = \frac{mv_f +\mu N t}{cos(\theta) t} \)

      • The only unknown is the normal forcee, but we can solve that by doing \( \Sigma{F_y} =0 \).

    • \( \Sigma{F_y} =0 ~\implies~ N - mg +Fsin(\theta) ~\implies~ N = mg-Fsin(\theta) \)

      • Plugging in normal force into F equation we get:

    • \( F = \frac{mv_f +\mu (mg-Fsin(\theta)) t}{cos(\theta) t} \)

      • Now we can solve for F.

    Level 3

      Solving for F we get:

    • F = \( \frac{m v_f}{cos(\theta)} + \frac{\frac{-sin(\theta)}{cos(\theta)^2} \frac{m v_f \mu}{t} + \frac{\mu mg }{cos(\theta)} }{1 + tan(\theta) \mu} \)
