Level 0

The first step is to find the time (t), when \( \theta = 53^o \). You can do this by solving the given function \( \theta = 4.2 t^{3/2} \) for t and plugging in \( \theta = 53^o. \)

t = 0.3647001689712002 seconds.

Level 1

Now that we have time t where \( \theta = 53^o \), we can differentiate both of the functions with respect to time and theta.

\( \theta = 4.2 t^{3/2} \)

  • Plugging in time to, \( \dot{\theta} \) and \( \dot{\dot{\theta} } \)
  • \( \dot{\theta} = \frac{ 4.2 (3)}{2}t^{1/2} \) = 3.804595866378838 \( \frac{rad}{s} \)
  • \( \dot{\dot{\theta} } = \frac{ 4.2 (3)}{4}t^{-1/2} \) = 5.216059917262172 \( \frac{rad}{s^2} \)

    • \( r = 4sin(2\theta) \) = 3.8450467837532756

  • Plugging in \( \dot{\theta} \) and \( \dot{\dot{\theta} } \) to \( \dot{r} \) and \( \dot{\dot{r} } \)
  • \( \dot{r} = 8cos(2\theta)\dot{\theta} \) = -8.389509956487581 \( \frac{m}{s} \)
  • \( \dot{\dot{r} } = -16sin(2\theta)\dot{\theta}^2+8cos(2\theta)\dot{\dot{\theta} } \) = -234.12936296238195\( \frac{m}{s^2} \)
    • Now that we have all of the values we can solve for velocity and acceleration.
