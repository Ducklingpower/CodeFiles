Level 0

To find the velocity of the slider, we have to use this equation of motion: \( v_s = v_o + \omega \) x \( r + v_{rel} \). Using this equation we can plug in what is given and solve for the velocity.

Level 1

Solving for the velocity using the equation: \( v_s = v_o + \omega \) x \( r + v_{rel} \).

  • \( v_s = 0 + \omega \) x \( [Lcos(\theta)\hat{i}-Lsin(\theta)\hat{j}]+vcos(\theta) \hat{i} - vsin(\theta)\hat{j} \) \( \implies \) \( \omega L cos(\theta)\hat{j}+\omega L sin(\theta)\hat{i} +vcos(\theta) \hat{i} - vsin(\theta)\hat{j} \)

    • *re-organizing*

  • \( \vec{v_s} = [vcos(\theta)+L\omega sin(\theta)]\hat{i} + [L\omega cos(\theta)-vsin(\theta)]\hat{j} \)
  • now to find the magnitude it is simply, \( v_s = \sqrt{(v_s\hat{i})^2 + (v_s\hat{j})^2} = 5.008991914547278 ~m/s \)

    • We can now solve for the acceleration. It is the same approach just with a different equation.

    Level 2

      To solve for the acceleration we can use the equation: \( A_s = A_o + \alpha \) x \( r-\omega^2r+2\omega \) x \( v_{rel} + A_{rel} \)

    • \( A_s = 0 + \alpha \)x \( [Lcos(\theta) - Lsin(\theta)]-\omega^2[Lcos(\theta)\hat{i} - Lsin(\theta)\hat{j}] + 2\omega \) x \( [vcos(\theta) - vsin(\theta)\hat{j}] + acos(\theta)\hat{i} + asin(\theta)\hat{j} \)

      • re-organizing and taking some cross products

    • \( A_s = [L\alpha sin(\theta) - \omega^2 L cos(\theta) + 2 \omega v sin(\theta) + a cos(\theta)] \hat{i} + [L\alpha cos(\theta) + \omega ^2Lsin(\theta) + 2\omega v cos(\theta) - a sin(\theta)]\hat{j} \)

      • Now, we can find the magnitude of the acceleration by doing,\( A_s = \sqrt{(A_s\hat{i})^2 + (A_s\hat{j})^2} = 10.323759005323593 ~m/s^2 \)
