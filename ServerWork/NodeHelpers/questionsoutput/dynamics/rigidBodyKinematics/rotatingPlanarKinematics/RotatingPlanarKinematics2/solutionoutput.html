Level 0

To find the velocity of the slider, we have to use this equation of motion: \( v = v_o + \omega \) x \( r + v_{rel} \). Using this equation we can plug in what is given and solve for the velocity.

Level 1

Solving for the velocity using the equation: \( v_s = v_o + \omega \) x \( r + v_{rel} \).

  • \( v_s = 0 + \omega \) x \( [0\hat{i}+ L\hat{j}]+ 0 \hat{i} + v\hat{j} \) \( \implies \) \( -l \omega \hat{i} + v \hat{j} \)
  • now to find the magnitude it is simply, \( v_s = \sqrt{(v_s\hat{i})^2 + (v_s\hat{j})^2} = 6.557438524302 ~m/s \)

    • We can now solve for the acceleration. It is the same approach just with a different equation.

    Level 2

      To solve for the acceleration we can use the equation: \( A_s = A_o + \alpha \) x \( r-\omega^2r+2\omega \) x \( v_{rel} + A_{rel} \)

    • \( A_s = 0 + \alpha \)x \( [0 \hat{i}+ L\hat{j}]-\omega^2[L\hat{j}] + 2\omega \) x \( [0 - v\hat{j}] + 0 \hat{i} + a\hat{j} \)

      • re-organizing and taking some cross products

    • \( A_s = [-L \alpha - 2\omega v ] \hat{i} + [-\omega ^2 L + a]\hat{j} \)

      • Now, we can find the magnitude of the acceleration by doing,\( A_s = \sqrt{(A_s\hat{i})^2 + (A_s\hat{j})^2} = 36.124783736376884 ~m/s^2 \)
