Level 0

Use the varibles given to compute the shear strain.

Level 1

The shear strain can be computed as \( \gamma = tan^{-1}( \frac{L_3}{L_1}) \)

Using \( \gamma \) we are able to calculate shear stress.

Level 2

    Shear stress:
  • \( \tau = G~\gamma = \frac{\tau_o}{\gamma_o}~ \gamma \)

    • We can use the Shear stress acting on the block to compute the force.
  • \( F = \tau~A = 9.525749593034567 \) N

Level 3

    To compute the permanent shear strain we can notice that the shear stress exceeded the elastic region and no longer acts elastically.
  • \( \gamma_{perm} = \gamma - \gamma_o = 0.00931152282607357 \)
