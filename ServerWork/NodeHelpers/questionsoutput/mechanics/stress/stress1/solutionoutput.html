Level 0

The first step is to draw a free-body diagram of the beam going from point B to C.

Level 1

The FBD will have a normal force pointing left coming from point B and a shear force coming from point B pointing up. There will be a moment reaction about point B and the distributed load pointing down on the beam. Before we are able to solve it, we have to figure out what the concentrated force is and where it is located.

Level 2

    Converting distributed load into one concentrated force: We can model this distributed load as a linear function. Using y=mx+b we can write our function as y=\( \frac{-F_d}{L_1+2L_1} \)x+\( F_d \). Using our function we can find the magnitude of the distributed load at point B. \( F_{dB} \) = \( \frac{-F_d}{L_1+2L_1}(L_1)+F_d \). Now we can find the concentrated force of the distributed load that goes from B to C. We can do this by calculating the area under the linear line. The area under the Line is equal to \( \frac {F_{dB}(2L_1)}{2} \). Lastly, we can compute the location of the concentrated force to be 1/3 the base.
  • Concentrated force = F = \( \frac{F_{dB}(2L_1)}{2} \)
  • Location of force = xx = \( \frac{2L_1}{3} \)
    • Now that we know the concentrated force we can compute the internal forces acting on point B

    Computing internal forces at point B using static equations

    Using the free body diagram:

  • \( \Sigma{F}_x = 0 ~\implies~ -N = 0 \)
  • N=0

  • \( \Sigma{F}_y = 0 ~\implies~ -F+V=0 \)
  • V = F = \( 188~ N \)

  • \( \Sigma{M}_B = 0 ~\implies~ M_B-F(xx) \)
