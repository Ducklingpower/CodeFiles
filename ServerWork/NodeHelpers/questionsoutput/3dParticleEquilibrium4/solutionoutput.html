Level 0

First you have to compute the vector expressions for the tensions \( T_A \) , \( T_B \) and \( T_C \). This can be done by multiplying the tension magnitude by the unit vector that follows the path of the tension.

Level 1

To compute the unit vector all you have to due is divide the length of the vector by its self.

Level 2

    Computing the tension vectors:
  • \( \vec{T}_A = (T_A)(\vec{U}_A) \). let the unit vector \( \vec{U}_A = \frac{0\hat{i}-L\hat{j}+0\hat{k}}{\sqrt{(L)^2}} = 0\hat{i}-1\hat{j}+0\hat{k} \)

    • So the tension vector \( \vec{T}_A = (T_A)(0\hat{i}-1\hat{j}+0\hat{k}) \)

  • \( \vec{T}_B = (T_B)(\vec{U}_B) \). let the unit vector \( \vec{U}_B = \frac{-L\hat{i}+0\hat{j}+0\hat{k}}{\sqrt{(L)^2}} = -1\hat{i}+0\hat{j}+0\hat{k} \)

    • So the tension vector \( \vec{T}_B = (T_B)(-1\hat{i}+0\hat{j}+0\hat{k}) \)

  • \( \vec{T}_C = (T_C)(\vec{U}_C) \). let the unit vector \( \vec{U}_C = \frac{L_1\hat{i}+L_2\hat{j}+L_3\hat{k}}{\sqrt{(L_1)^2+(L_2)^2+(L_3)^2}} \)

    • So the tension vector \( \vec{T}_D = (T_D)\frac{L_1\hat{i}+L_2\hat{j}+L_3\hat{k}}{\sqrt{(L_1)^2+(L_2)^2+(L_3)^2}} \)

  • Now that you have all of the force vectors, set up the static equilibrium force equations.

Level 3

    Setting up static equilibrium force equations:
  • \( \Sigma{F}_x= 0 \implies -\vec{T}_B\hat{i}+\vec{T}_C\hat{i}=0 \)
  • \( \Sigma{F}_y= 0 \implies -\vec{T}_A\hat{j}+\vec{T}_C\hat{j}=0 \)
  • \( \Sigma{F}_z= 0 \implies \vec{T}_C\hat{k} - W = 0 \)

    • Now you can solve for the tension in each cable.

Level 4

    Solving force equations:

  • \( {T}_C = \frac{W}{\hat{k}} \)
    • Where \( \hat{k} \) is the k component of \( \vec{U}_C \)

    Now we can solve for tension A and B:

  • \( T_B = \vec{T}_C\hat{i} \)
  • \( T_A =\vec{T}_C\hat{j} \)
    • Now that we have tension A and B, we can find the stretch in the springs.
