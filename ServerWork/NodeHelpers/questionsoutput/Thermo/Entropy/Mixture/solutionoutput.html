Level 2

Using the First Law of Thermodynamics for a steady-state open system:

\( \dot{Q} + \dot{m}_1(h_1 + ke_1 + pe_1) + \dot{m}_2(h_2 + ke_2 + pe_2) = \dot{W} + \dot{m}_3(h_3 + ke_3 + pe_3) \)

Assumptions that allow us to simplify the equation:

  • The system is adiabatic, so \( \dot{Q} = 0 \).
  • There is no shaft work, so \( \dot{W} = 0 \).
  • Changes in kinetic energy are negligible: \( ke_1 = ke_2 = ke_3 = 0 \).
  • Changes in potential energy are negligible: \( pe_1 = pe_2 = pe_3 = 0 \).

Thus, the equation simplifies to:

\( \dot{m}_1 h_1 + \dot{m}_2 h_2 = \dot{m}_3 h_3 \)

Solving for \( \dot{m}_1 \):

\( \dot{m}_1 = \frac{\dot{m}_2(h_3 - h_2)}{h_1 - h_3} \)

Substituting values:

\( \dot{m}_1 = 4.082 \) kg/s

And the total mass flow rate:

\( \dot{m}_3 = \dot{m}_1 + \dot{m}_2 = 69.282 \) kg/s

Level 3

Using the Second Law of Thermodynamics for entropy balance:

\( \frac{dS_{cv} }{dt} = \sum \frac{\dot{Q}_i}{T_i} + \sum \dot{m}_{in} s_{in} - \sum \dot{m}_{out} s_{out} + \dot{S}_{prod} \)

Since it is adiabatic and steady-state, this reduces to:

\( \dot{m}_3 s_3 - \dot{m}_1 s_1 - \dot{m}_2 s_2 = \dot{S}_{prod} \)

Substituting values:

\( \dot{S}_{prod} = (69.282 kg/s)(1.503 kJ/kg-K) - (4.082 kg/s)(7.154 kJ/kg-K) - (65.2 kg/s)(1.134 kJ/kg-K) \)
