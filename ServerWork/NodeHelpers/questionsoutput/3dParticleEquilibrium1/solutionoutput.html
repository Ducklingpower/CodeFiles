Level 2

    Computing the Force vectors:
  • \( \vec{F}_O = (F_O)(\vec{U}_O) \). let the unit vector \( \vec{U}_O = \frac{-L_3\hat{i}-L_1\hat{j}+L_1\hat{k}}{\sqrt{(L_3)^2+(L_1)^2+(L_1)^2}} \)

    • So the force vector \( \vec{F}_O = (F_O)\frac{-L_3\hat{i}-L_1\hat{j}+L_1\hat{k}}{\sqrt{(L_3)^2+(L_1)^2+(L_1)^2}} \).

  • \( \vec{F}_P = (F_P)(\vec{U}_P) \). let the unit vector \( \vec{U}_P = \frac{-L_2\hat{i}+L_2\hat{j}+0\hat{k}}{\sqrt{(L_2)^2+(L_2)^2}} \) .

    • So the force vector \( \vec{F}_P = (F_P) \frac{-L_2\hat{i}+L_2\hat{j}+0\hat{k}}{\sqrt{(L_2)^2+(L_2)^2}} \).

  • \( \vec{F}_B = (F_B)(\vec{U}_B) \). let the unit vector \( \vec{U}_B = \frac{L_2\hat{i}+L_1\hat{j}+L_2\hat{k}}{\sqrt{(L_2)^2+(L_2)^2+(L_1)^2}} \) .

    • So the force vector \( \vec{F}_B = (F_B) \frac{L_2\hat{i}+L_1\hat{j}+L_2\hat{k}}{\sqrt{(L_2)^2+(L_2)^2+(L_1)^2}} \)

  • \( \vec{F}_R = (F_R)(\vec{U}_R) \). let the unit vector \( \vec{U}_R = \frac{L_2\hat{i}-L_2\hat{j}-L_3\hat{k}}{\sqrt{(L_2)^2+(L_2)^2+(L_3)^2}} \) .

    • So the force vector \( \vec{F}_R = (F_R) \frac{L_2\hat{i}-L_2\hat{j}-L_3\hat{k}}{\sqrt{(L_2)^2+(L_2)^2+(L_3)^2}} \)

  • Now that you have all of the force vectors, set up the static equilibrium force equations.

Level 3

    Setting up static equilibrium force equations:
  • \( \Sigma{F}_x= 0 \implies -\vec{F}_O\hat{i}-\vec{F}_P\hat{i}+\vec{F}_B\hat{i}+\vec{F}_R\hat{i}+\vec{F}\hat{i} = 0 \)
  • \( \Sigma{F}_y= 0 \implies -\vec{F}_O\hat{j}+\vec{F}_P\hat{j}+\vec{F}_B\hat{j}-\vec{F}_R\hat{j}+\vec{F}\hat{j} = 0 \)
  • \( \Sigma{F}_z= 0 \implies \vec{F}_O\hat{z}+\vec{F}_B\hat{z}-\vec{F}_R\hat{z}+\vec{F}\hat{z} = 0 \)

    • Where \( \vec{F} \) is the messing vector we want to find.

    Solving each equation for \( \vec{F}\hat{i} \),\( \vec{F}\hat{j} \), and \( \vec{F}\hat{k} \)

  • \( \vec{F}\hat{i} = \vec{F}_O\hat{i}+\vec{F}_P\hat{i}-\vec{F}_B\hat{i}-\vec{F}_R\hat{i} \)
  • \( \vec{F}\hat{j} = \vec{F}_O\hat{j}+\vec{F}_R-\vec{F}_P\hat{j}\hat{j}-\vec{F}_B\hat{j} \)
  • \( \vec{F}\hat{k} = -\vec{F}_O\hat{k}-\vec{F}_B\hat{k}+\vec{F}_F\hat{k} \)
    • Now that you have the missing vector, compute the magnitude and the unit vector.
