Level 0

To determine the amount that should be deposited at time \( t_0 \) into the fund, we need to calculate the present value of each withdrawal and sum these values.

Level 1

The present value of each withdrawal can be calculated using the formula: \[ PV = \frac{FV}{(1 + r)^n} \] where \( FV \) is the future value (withdrawal amount), \( r \) is the interest rate per period, and \( n \) is the number of periods until the withdrawal.

Level 2

For the three withdrawals, we calculate: Here \( t_0 = 0 \)

  • For the first withdrawal of \( \ 1449 \) at time \( t_1 \): \[ PV_1 = \frac{ 1449 }{(1 + 0.05)^{t_1 - t_0} } \]
  • For the second withdrawal of \( \ 4044 \) at time \( t_2 \): \[ PV_2 = \frac{ 4044 }{(1 + 0.05)^{t_2 - t_0} } \]
  • For the third withdrawal of \( \ 2178 \) at time \( t_3 \): \[ PV_3 = \frac{ 2178 }{(1 + 0.05)^{t_3 - t_0} } \]

Level 3

The total present value (deposit amount) required at time \( t_0 \) is the sum of the present values of all withdrawals: \[ Deposit = PV_1 + PV_2 + PV_3 \]

Level 4

Substituting the values gives us the required deposit amount to ensure the fund is depleted at the last withdrawal: \[ Deposit = \frac{ 1449 }{(1 + 0.05)^{t_1 - t_0} } + \frac{ 4044 }{(1 + 0.05)^{t_2 - t_0} } + \frac{ 2178 } {(1 + 0.05)^{t_3 - t_0} } \]
