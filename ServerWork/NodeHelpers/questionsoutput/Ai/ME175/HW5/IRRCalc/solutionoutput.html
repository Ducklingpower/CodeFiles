Pw = -(Initial Investment) + (Cash flow) + (Present given future)

  • Where the initial investment is 34945

  • The cash flow is given as: \( 7100 \cdot \frac{(1+0.15)^{ 9 }-1 } { 0.15(1 + 0.15)^{ 9 } } \)

  • The Present given future can be written as: \( \frac{ 34945 }{ (1+ 0.15 )^{ 28 } } \)

    Pw = -34945 + \( 7100 \cdot \frac{(1+0.15)^{ 9 }-1 } { 0.15(1 + 0.15)^{ 9 } } \) + \( \frac{ 34945 }{ (1+ 0.15 )^{ 28 } } \) = -368.77 USD
