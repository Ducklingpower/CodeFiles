Level 0

To calculate the equivalent present worth (PW) of the cash flows, we can use the formula: \[ PW = \sum_{t=0}^{n} \frac{R_t}{(1 + i)^t} \] where \( R_t \) is the net cash flow in year \( t \), \( i \) is the MARR, and \( n \) is the useful life of the asset.

Level 1

First, we need to determine the net cash flow for each year: \[ R_t = \text{Annual Income} - \text{Annual Expenses} \] Substitute the known variables: \[ R_t = 8753 - 2148 \]

Level 2

Next, we will calculate the present worth for the income generated for each year over the lifespan of the band saw, excluding the salvage value, using the MARR: \[ PW_{income} = \sum_{t=1}^{n} \frac{R_t}{(1 + \frac{ 5 }{100})^t} = (8753 - 2148) \frac{(1 + \frac{ 5 }{100})^n -1}{ \frac{ 5 }{100}(1 + \frac{ 5 }{100})^n } \] and for the salvage value at the end of the lifespan: \[ PW_{salvage} = \frac{ 6000 }{ {(1 + \frac{ 5 }{100})^{n} } } \]

Level 3

Finally, the total equivalent present worth is given by: \[ PW = -Cost PW_{income} + PW_{salvage} \] Substitute the values to calculate the final present worth: \[ PW =- 17000 + (8753 - 2148) \frac{(1 + \frac{ 5 }{100})^n -1}{ \frac{ 5 }{100}(1 + \frac{ 5 }{100})^n } + \frac{ 6000 }{ {(1 + \frac{ 5 }{100})^{n} } } \]

Level 4

Therefore, the equivalent present worth of the cash flows is \( PW = 25483.08 ~USD \).

Level 5

The annual worth is calculated in a simular way, using annual given present:

A = pw \( \frac{i(1+i)^n}{(i+1)^n -1 } \)

Plugging in we can solve for A:

\[ A = 25483.08 \frac{ 0.05(1+0.05)^{7} }{(1+0.05)^{7} -1 } = 4403.982~ USD \]

Level 6

The future worth can be solved by bringing the present worth fowards in time: Fw = Pw\( (1+i)^7 \)

Plugging in we can solve for the future worth to be:

\[Fw = 25483.08 (1+0.05)^7 = 35857.258 ~USD \]

Level 7

To solve for the internal rate of return (IRR) we have to set the present worth equal to zero and solve for i = irr.

\[ PW= 0 =- 17000 + (8753 - 2148) \frac{(1 + IRR)^n -1}{ IRR(1 + IRR)^n } + \frac{ 6000 }{ {(1 + IRR)^{n} } } \]
