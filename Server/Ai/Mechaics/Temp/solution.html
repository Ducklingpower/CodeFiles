To determine the temperature at which the rail will buckle, we need to calculate the stress caused by thermal expansion and set it equal to the buckling stress.

The stress due to thermal expansion is given by:

  • $\sigma_{thermal} = E \cdot \alpha \cdot \Delta T$

Where:

  • $E$ is the modulus of elasticity, given as ${{params.modulusElasticity}} \times 10^{11} {{params.unitsModulus}}$
  • $\alpha$ is the coefficient of thermal expansion, given as ${{params.thermalExpansionCoeff}} \times 10^{-6} {{params.unitsAlpha}}$
  • $\Delta T$ is the change in temperature from the stress-free temperature $T$.

Equate the thermal stress and the buckling stress $\sigma$ to solve for the change in temperature $\Delta T$:

  • $E \cdot \alpha \cdot \Delta T = {{params.stress}} {{params.unitsStress}}$
  • Solve for $\Delta T$:
  • $\Delta T = \frac{ { {{params.stress}} } } { {E \cdot \alpha} }$

Since we need to find the buckling temperature $T_{buckling}$, add this change in temperature to the stress-free temperature:

  • $T_{buckling} = T + \Delta T$
  • Where $T = {{params.tempFree}} {{params.unitsTemp}}$ is the initial stress-free temperature.

Now substitute the known values into the equations to find the numerical solution:

  • Calculate $\Delta T$ using:
  • $\Delta T = \frac{ { { {params.stress} } } } { { {{params.modulusElasticity}} \times 10^{11} \times {{params.thermalExpansionCoeff}} \times 10^{-6} } }$
  • Finally determine $T_{buckling}$:
  • $T_{buckling} = {{params.tempFree}} + \Delta T$
