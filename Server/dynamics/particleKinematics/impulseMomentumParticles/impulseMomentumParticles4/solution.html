To find the final velocity of the block and bullet, use the conservation of moment equation.

Conservation of momentum:

  • $v_B m_B + v m = (m_B + m ) v_f ~\implies ~ \frac{v_B m_b + 0 }{m_b + m} = v_f ={{correct_answers.v}}~m/s$

    • Where $v_f$ is the velocity of the block and the bullet after the collision.

    To find the time, compute the impulse-momentum principle.

      Impulse momentum principle, $m v_1 + \Sigma \int F dt = m v_2$

      Applying in the Y direction:

    • $ 0 + (N (t) + mg (t)) =0 ~\implies~ N = mg$

      • Applying in the X direction:

    • $m v_f - (\mu N t) = 0 ~implies~ t = \frac{m v_f}{\mu N}$

      • Plugging in N we can solve for t:

    • $t = \frac{m v_f}{\mu (mg)} ~\implies~ \frac{v_f}{\mu g} = {{correct_answers.t}} ~s$

      • Now we can find the total distance traveled.

      Using the kinematic equation : $v_f^2 = v_i ^2 -2 a \Delta x$. we can aptian the distance traveled.

    • $0 = v_i ^2 -2 a \Delta x ~\implies~ \Delta x = \frac{v_i ^2}{2a}$

      • Here the deceleration is $a = \frac{v_f}{t}$, the speed of the block divided by the time it takes the block to come to a rest. We can now plug that into our equation. Also here the initial velocity is equal to $v_f$. So $v_f = v_i$

    • $\Delta x = \frac{v_i ^2}{2(\frac{v_f}{t})} ~\implies ~ \Delta x = \frac{v_f t}{2} = {{correct_answers.s}}~m$
