The first step is to define the datum so that you can write the position equation for the rope.

If we define the datum at the top of the pulley system we can come up with the equation for the length of the rope as: $s_A + 4 s_B = L$

Taking the time derivative we get: $v_A + 4 v_B = 0$, lets hold on to this equation.

  • $v_A + 4 v_B = 0$

    • We can write the impulse-momentum equation for each block we get:

  • $0 + (m_A g (t) - T(t)) = m_A v_A$
  • $0 + (m_B g (t) - 4T(t)) = m_B v_B$

    • Now we have 3 equations and 3 unknowns.

      Solving for $v_B$ we get:

    • $v_B = -gt^2\frac{4 m_A - m_B}{16 m_A t + m_B t} = {{correct_answers.vb}} ~m/s$
