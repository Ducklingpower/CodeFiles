The first step is to find the time (t), when $\theta = {{params.theta}}^o$. You can do this by solving the given function $\theta = {{params.c}} t^{3/2}$ for t and plugging in $\theta = {{params.theta}}^o.$

t = {{params.t}} seconds.

Now that we have time t where $\theta = {{params.theta}}^o$, we can differentiate both of the functions with respect to time and theta.

$\theta = {{params.c}} t^{3/2}$

  • Plugging in time to, $\dot{\theta}$ and $\dot{\dot{\theta} }$
  • $\dot{\theta} = \frac{ {{params.c}} (3)}{2}t^{1/2}$ = {{params.theta1}} $\frac{rad}{s}$
  • $\dot{\dot{\theta} } = \frac{ {{params.c}} (3)}{4}t^{-1/2}$ = {{params.theta2}} $\frac{rad}{s^2}$

    • $r = 4sin(2\theta)$ = {{params.r}}

  • Plugging in $\dot{\theta}$ and $\dot{\dot{\theta} }$ to $\dot{r}$ and $\dot{\dot{r} }$
  • $\dot{r} = 8cos(2\theta)\dot{\theta}$ = {{params.r1}} $\frac{m}{s}$
  • $\dot{\dot{r} } = -16sin(2\theta)\dot{\theta}^2+8cos(2\theta)\dot{\dot{\theta} }$ = {{params.r2}}$\frac{m}{s^2}$
    • Now that we have all of the values we can solve for velocity and acceleration.
      Solving for velocity and acceleration:

      $V = \sqrt{(\dot{r})^2+(r\dot{\theta})^2}$ = {{correct_answers.V}} $\frac{m}{s}$

      $A = \sqrt{(\dot{\dot{r} }-r\dot{\theta}^2)^2+(r\dot{\dot{\theta} }+2\dot{r}\dot{\theta})^2}$ = {{correct_answers.A}} $\frac{m}{s^2}$
