The first step is to identify that there are two components to the acceleration. The first is a tangential acceleration and the second is a normal acceleration.

Solving for tangential acceleration:

Acceleration tangent to the surface the ball is sliding on is defined as $g~sin(\theta)$

Computing theta: By definition, the derivative of (y) gives us the tangential slope. We can use this slope to find theta.

  • $\frac{dy}{dx} =2 c_1 x$. Plugging in $L_1$ for x we get a slope of $\frac{1}{c_1} L_1$.
  • So, $\theta = tan^{-1}(\frac{1}{c_1} L_1) $

    • So now plugging in theta we have tangential acceleration, $a_t = g~sin(\theta)$

    Now we must solve for normal acceleration.

      The normal acceleration is defined to be : $\frac{v^2}{r}$

      Solving for r:

    • $r = \frac{(1 + (\frac{dy}{dx})^2)^{\frac{3}{2} } } { \frac{d^2y}{dx^2} }$ $~\implies~$ $r =\frac{(1+(\frac{2 L_1}{c_1})^2 )^{\frac{3}{2} } }{\frac{2}{c_1} }$

      • Now we have normal acceleration: $a_n = \frac{v^2}{r}$

      Now all that's left to do is take the magnitude of the two accelerations.

      Magnitude of acceloration = $\sqrt{(a_n)^2 + (a_t)^2}$

    • $a_{net} = \sqrt{(\frac{v^2}{r})^ + (g~sin(\theta))^2}$ = {{correct_answers.a}} $m/s^2$
