This is a curvilinear problem, we can take the derivatives of the theta and r equation to obtain the acceleration and velocity in the theta and r direction.

Talking derivatives of r:

  • $r = c_1 t^2$
  • $\dot{r} = 2c_1 t$
  • $\dot{\dot{r} } = 2c_1$

    • Taking derivatives of $\theta$

  • $\theta = c_2 t^3$
  • $\dot{\theta} = 3c_2 t^2$
  • $\dot{\dot{\theta} } = 6c_2t$

    • Now we can solve for velocity and acceleration.

      Solving for Velocity:

    • $v = \sqrt{ (\dot{r})^2 + (r \dot{\theta})^2 }$
    • $v = \sqrt{ (2c_1 t)^2 + ((c_1 t^2) ~ 3c_2 t^2)^2 }$ = {{correct_answers.v}} m/s

      • Solving for acceleration:

    • a =$ \sqrt{(\dot{\dot{r} } - r\dot{\theta}^2)^2 + (r \dot{\dot{\theta} } + 2 \dot{r} \dot{\theta})^2}$
    • a =$ \sqrt{(2c_1 - c_1t^2 (c_2 3 t^2)^2 )^2 + (c_1 t^2 (6c_2 t ) + 2 (2c_1t) c_2 3 t^2)^2}$ = {{correct_answers.a}} $m/s^2$
