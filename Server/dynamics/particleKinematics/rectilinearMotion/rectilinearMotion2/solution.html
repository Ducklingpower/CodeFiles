First seperate the problem into two compenets. Up and down and left and right. In the Horizontal direction, the velocity will be constant, and in the verticle direction, the acceleration is constant pointing downwards.

Writing the equations of motion for both directions we get:

  • Horizontal: $V = \frac{D}{\Delta t}$ $\implies$ $V_o cos(\theta) = \frac{D}{\Delta t}$

  • Verticle: $ \Delta S = V_o t + \frac{1}{2}gt^2$ $\implies$ $0 = V_osin(\theta)t - \frac{1}{2}gt^2$ $\implies$ $2v_o = gt$
    • When breaking up the problem into separate components, it's important to note that they will also be connected by time. So we can use that to calculate theta.
    • Horizontal: $V = \frac{D}{\Delta t}$ solving for time $t = \frac{D}{V_ocos(\theta)}$

      • Now we can plug in time into the Verticle equation

    • Verticle: $\frac{2V_osin(\theta)}{g} = \frac{D}{V_ocos(\theta)}$ $\implies$ $sin(\theta)cos(\theta) = \frac{Dg}{2V_o^2}$ $\implies$ $ \frac{1}{2}sin(2\theta) = \frac{Dg}{2V_o^2}$ $\implies$ $\theta = \frac{1}{2}sin^{-1}(\frac{Dg}{V_o^2})$
      • Solution: $\theta = \frac{1}{2}sin^{-1}(\frac{ {{params.D}} (9.81)}{ {{params.V}} ^2}) = {{correct_answers.theta}}^o$
      To find the max distance the ball can travel with an initial speed of $v_o = {{params.V}}$ all you have to do is set $\theta$ to $45^o$

      Using the equation from above, $\frac{1}{2}sin(2\theta) = \frac{Dg}{2V_o^2}$, we can solve for maximum distance . $D = sin(2\theta)\frac{V_o^2}{g} $

      Solution: When $\theta = 45$: $D_{max} = sin(90^o)\frac{ {{params.V}}^2}{9.81} = {{correct_answers.Dmax}}$ meters
