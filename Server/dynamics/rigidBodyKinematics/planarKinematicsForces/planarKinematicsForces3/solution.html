The first step is to draw the free-body diagram. We can compute the angular acceleration by doing a moment balance equation about G.

    Moment equation:

  • $\Sigma {M_G} = I_g \alpha ~\implies~ I_G \alpha = F(L_3)$
    • K = $\sqrt{\frac{I}{m} }$, so $I = m k^2$

    Plugging in I and solving for alpha we get:

  • $\alpha = \frac{F (L_3) }{k^2 m}$ = {{correct_answers.alpha}} $rad/s^2$
