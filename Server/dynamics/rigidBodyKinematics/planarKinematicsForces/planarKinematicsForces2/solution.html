The first step is to draw a free-body diagram.

Free body diagram:

  • Weight force pointing down
  • Friction force pointing up
  • Normal force pointing right
  • Tension force points up along the support beam

    • Now you can write out the force equations.

      Force equations:

    • $\Sigma{F_x} = 0 ~\implies~ Tcos(\theta) - N = 0 $
    • $\Sigma{F_y} = 0 ~\implies~ F_{fric} - w + Tsin(\theta)= 0 ~\implies~ \mu N - w + Tsin(\theta)= 0$
    • $\Sigma{M_G} = I_g \alpha ~\implies~ F_{fric}r= I_g \alpha$

      • Here you have 3 equations and 3 unknowns: (T,N,$\alpha$)

      Now you can try and solve for them

      Solving the equations we can obtain the unknowns:

    • $T = \frac{w}{\mu cos(\theta) + sin(\theta)} $ = {{correct_answers.T}} N
    • $F_{fric} = w - Tsin(\theta)$ = {{correct_answers.fric}} N
    • $\alpha = \frac{F_{fric} r }{I} ~\implies~ \frac{2F_{fric} }{mr}$ = {{correct_answers.alpha}} $rad/s^2$

      • Here I = $\frac{1}{2}mr^2$

      Now that we have the deceleration of the system ($\alpha$). We can compute time to slow down as: t = $\frac{\omega}{\alpha}$ = {{correct_answers.t}} s
