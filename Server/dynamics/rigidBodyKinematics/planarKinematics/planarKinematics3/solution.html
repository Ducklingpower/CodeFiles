First we can write out the equation of acceleration at B to be: $\vec{a_B} = \vec{a_A} + \alpha$ x $\vec{L} -\omega^2L $

Here we have everything we need to find acceleration and angular acceleration other than $\omega$. So we have to find that first.

 Solving for $\omega$:
  • $\vec{v_B} = \vec{v_A} + \omega$ x $\vec{L}$

    • Plugging in $\vec{v_A}$ and taking the cross product we get:

  • $\vec{v_B} = -v~\hat{j} + Lcos(\theta)\omega ~\hat{j} + Lsin(\theta)\omega ~ \hat{i}$

    • We can break up $\vec{v_B}$ into $\hat{i} ,\hat{j}$ components to solve for $\omega$

  • $v_B\hat{i} =Lsin(\theta)\omega ~ \hat{i}$
  • $0 = -v~\hat{j} + Lcos(\theta)\omega ~\hat{j}$

    • Solving for $\omega$ we get:

  • $\omega$ = $\frac{v}{Lcos(\theta)}$

    • Now we can solve for angular acceleration and normal acceleration at B.
    • $\vec{a_B} = \vec{a_A} + \alpha$ x $\vec{L} -\omega^2L $

      • We can expand this equation further by taking the cross-product.

    • $\vec{a_B} = -a~\hat{j} + Lcos(\theta)\alpha~\hat{j} + Lsin(\theta)\alpha~\hat{i} - \omega^2Lcos(\theta)~\hat{i} + \omega^2Lsin(\theta)~\hat{j}$

      • Now we can break up $\vec{a_B}$ into components and solving for $\alpha$ and acceleration

      Solving for $\alpha$ and acceleration .

    • 0$\hat{j} = -a + \omega^2Lsin(\theta) + Lcos(\theta)\alpha$ $\implies$ $\alpha = \frac{a-[\frac{v}{Lcos(\theta)}]^2Lsin(\theta)}{Lcos(\theta)}$ = {{correct_answers.alpha}} $rad/s^2$

    • $a\hat{i} = Lsin(\theta)\alpha - \omega^2Lcos(\theta) $ $\implies$ $ Lsin(\theta)[\frac{a-[\frac{v}{lcos(\theta)}]^2Lsin(\theta)}{Lcos(\theta)}] - [\frac{v}{lcos(\theta)}]^2Lcos(\theta)$ = {{correct_answers.acc}} $m/s^2$
