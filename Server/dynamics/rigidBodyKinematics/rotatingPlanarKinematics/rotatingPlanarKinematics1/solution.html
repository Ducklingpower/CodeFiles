To find the velocity of the slider, we have to use this equation of motion: $v_s = v_o + \omega $ x $ r + v_{rel}$. Using this equation we can plug in what is given and solve for the velocity.

Solving for the velocity using the equation: $v_s = v_o + \omega $ x $ r + v_{rel}$.

  • $v_s = 0 + \omega $ x $ [Lcos(\theta)\hat{i}-Lsin(\theta)\hat{j}]+vcos(\theta) \hat{i} - vsin(\theta)\hat{j}$ $\implies$ $\omega L cos(\theta)\hat{j}+\omega L sin(\theta)\hat{i} +vcos(\theta) \hat{i} - vsin(\theta)\hat{j} $

    • *re-organizing*

  • $\vec{v_s} = [vcos(\theta)+L\omega sin(\theta)]\hat{i} + [L\omega cos(\theta)-vsin(\theta)]\hat{j}$
  • now to find the magnitude it is simply, $v_s = \sqrt{(v_s\hat{i})^2 + (v_s\hat{j})^2} = {{correct_answers.velocity}} ~m/s$
