In this problem you can solve it by using conservation of angular momentum.

    Using conservation of linear moment on the pully:

  • In state 1, the pully is at rest so it has no momentum.
  • in state 2, the pull is rotating with momentum after time has passed.

    • So applying these states we get:

  • $0 + ( F r_1 (t) - m_A ~g~ r_2 (t) ) = I \omega + r_2 m_p v_A$

    • We can replace $\omega$ with ($\frac{v_A}{r_2}$)

  • $0 + ( F r_1 (t) - m_A~ g~ r_2 (t) ) = I \frac{v_A}{r_2} + r_2 m_p v_A$

    • Now we can solve for $v_A$.

    Solving for the velocity of the block we get:

  • $v_A = \frac{F r_1 (t) - m_A~ g~ r_2 (t)}{I \frac{1}{r_2} + r_2 m_A}$

    • $I = m_p k^2$

  • $v_A = \frac{F r_1 (t) - m_A ~g~ r_2 (t)}{m_p k^2 \frac{1}{r_2} + r_2 m_A}$ = {{correct_answers.v}} $m/s$
