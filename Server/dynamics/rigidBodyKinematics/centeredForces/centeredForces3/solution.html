The first step is to draw a free-body diagram of the wheel.

The only forces acting on the wheel are the weight force and the external force F.

Equation of motion:

  • $\Sigma{M_G} = I\alpha ~\implies~ I\alpha = -rF ~\implies~ \alpha = \frac{-rF}{I} $

    • We can compute inertia a $I = k^2m$

  • So, $\alpha = \frac{-rF}{k^2m}$

    • Now that we have angular acceleration we can use kinematics to solve for angular velocity

      Solving for $\omega$

      $A = \alpha r$

      $v^2_f = v^2_i + 2 A \Delta x$ . But $v^2_i$ is zero so, $ v_f = \sqrt{2 A \Delta x }$

    • lastly $\omega = \frac{v_f}{r}$
    • Doing it all at once we have $v_f = \frac{\sqrt{2\frac{r^2Fg}{k^2 w } \Delta x } }{r} = {{correct_answers.w}} ~\frac{rad}{s}$
